# Facilython
Una forma de que todos aprendan a programar.
# Por que usar Facilython?
Facilython es una capa de transpilacion de codigo para el lenguaje de programacion Python.

Este proyecto esta diseñado para aquellos que se les dificulta aprender Python (ya sea por que esta en ingles, es muy complicado, etc.), asi que si no sabes usar Python este proyecto es para ti, ya que es facil de entender y esta en español.
# Como instalar?
1) Clonar este repositorio
```
git clone https://github.com/SoloUnRandomX/facilython
```
2) Programar tu app, aqui esta una lista de las binds actuales:
```
si = if
sino = else
imprimir = print
mientras = while
para = for
en = in
definir = def
retornar = return
```
3) Ejecutar el archivo **exec.py**
4) Y listo!
